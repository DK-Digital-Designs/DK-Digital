GoalGetters

A community-driven platform engineered for motivation and growth.

Live Demo GitHub Repo

GoalGetters was designed to help users stay motivated by tracking their progress within a supportive community. We focused on highly-efficient state management and visual encouragement to keep users engaged.

The Stack

  • Vite
  • Vanilla JavaScript
  • CSS Variables
  • Local Storage

Cool Features

  • Dynamic goal tracking with progress visualizers.
  • Community-led motivation boards.
  • Strict focus on core web vitals and performance.
  • Dark/Light mode persistent syncing.
